Реализован полноценный раздел "Новости" для портала "Здоровый стол"

- Добавлена страница со списком новостей:

  • Поддержка поиска по ключевым словам

  • Фильтрация по категориям

  • Сортировка по дате публикации и популярности

- Создана детальная страница отдельной статьи с полным отображением контента

- Расширен функционал Django Admin для работы с новостями:

  • Удобное создание и редактирование статей

  • Интеграция django-ckeditor для богатого текстового редактора (поддержка форматирования, изображений, таблиц и т.д.)

- Интеграция новостного блока на главную страницу с автоматическим выводом последних статей в разделе Новые статьи
